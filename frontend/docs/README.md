# Sistema de Gestão de OS - Dashboard

Projeto Next.js 14+ com TypeScript, Tailwind CSS e App Router para gestão operacional, CRM, vendas e inteligência de um sistema de Ordens de Serviço.

## Requisitos

- Node.js 18+ instalado

## Instalação

```bash
npm install
npm run dev
```

## Estrutura principal

- `app/` - App Router, páginas e layout principal com Sidebar
- `components/` - Componentes reutilizáveis (Sidebar, UI, templates, modais)
- Persistência em `localStorage` com automações de OS → Fatura → Comissão
